# Sharon Weds Amala

My brother Sharon is getting married, so I built him a wedding RSVP website he can drop into WhatsApp groups. Guests can RSVP, see event details, check venues on a map, and add the wedding to their Google Calendar — all in one page.

Link: [sharonwedsamala.vercel.app](https://sharonwedsamala.vercel.app)

## Stack

React 19 · TypeScript · Vite · Framer Motion · Supabase · Vercel · OpenStreetMap

## Run Locally

```bash
npm install
npm run dev
```

Requires `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY` in `.env`. Run `supabase/schema.sql` in your Supabase project SQL Editor.
